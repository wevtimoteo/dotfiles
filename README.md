# Dotfiles for MacOS

Also check the [Linux version](https://github.com/wevtimoteo/dotfiles/tree/linux).

## Setup

```
git clone git://github.com/wevtimoteo/dotfiles.git
cd dotfiles

./install.zsh --simulate

brew bundle
```

Omit `--simulate` to actually run it.
